🖨 1번 컴퓨터 다이렉트 출력 안내

1번 컴퓨터에서는 결제 후 바로 출력됩니다. 핀번호 없이 사용할 수 있습니다.

📋 출력 전 확인 사항

화면에 출력 리스트 박스(CLOPY)가 떠 있어야 합니다.

⚠️ 출력 리스트 박스가 안 보일 때

화면 오른쪽 하단 화살표(▲)를 누르면 숨겨진 아이콘이 나옵니다. 그중 노란 아이콘(KSPCCLient)을 더블 클릭하면 출력 프로그램이 다시 실행됩니다.

📌 출력 순서

  1. 1
    프로그램에서 인쇄 버튼을 누른 뒤, 흑백/컬러 등 옵션을 설정하고 인쇄 버튼을 클릭합니다.
  2. 2
    출력 리스트에 작업이 올라옵니다. 출력 매수와 금액을 확인하세요.
  3. 3
    출력비용결제 / 출력 버튼을 눌러 결제를 진행합니다.
  4. 4
    결제 — 신용카드, 삼성페이, 카카오페이, 제로페이, 티머니 등으로 결제합니다. 카드는 화면 아래 슬롯에 넣거나, 화면에 대서 터치합니다.
  5. 5
    출력물 확인 — 복합기 출력구에서 인쇄물을 가져가세요.
