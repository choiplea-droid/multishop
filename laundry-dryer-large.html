건조기 대형 사용안내

🚫 이용 불가 안내

  • 🔴 애완동물 용품 건조 불가
  • 위반 시 손해배상 및 기기 클리닝 비용이 청구될 수 있습니다.

📌 기본 정보

  • 건조기 종류: 대형
  • 요금: 500원 투입 시 4분 추가
  • 예시: 2,000원 → 16분 / 4,000원 → 32분

✔ 원하는 시간만큼 금액을 투입하시면 됩니다.

🌡 온도 선택 안내

옷감에 따라 온도를 선택해주세요.

  • 🔹 저온 → 섬세의류, 기능성 의류
  • 🔸 중온 → 일반 의류(티셔츠, 나일론 등)
  • 🔺 고온 → 수건, 두꺼운 의류

✔ 옷감에 맞게 온도와 시간을 조절해주세요.

💰 결제 방법

💵 현금 결제

⚠️ 사용하려는 건조기 번호/투입구가 맞는지 반드시 확인해주세요.

  1. 1
    세탁물 투입 후 문을 완전히 닫기
  2. 2
    500원 동전 투입 (투입 금액만큼 시간 증가)
  3. 3
    저온, 중온, 고온 중에서 온도 선택
  4. 4
    START 버튼을 눌러 건조 시작
💳 선불카드 결제

⚠️ 사용하려는 건조기 번호/투입구가 맞는지 반드시 확인해주세요.

  1. 1
    세탁물 투입 후 문을 완전히 닫기
  2. 2
    카드 태그기로 500원 단위 결제 (1회 태그 시 500원 차감 → 4분 추가)
  3. 3
    저온, 중온, 고온 중에서 온도 선택
  4. 4
    START 버튼을 눌러 건조 시작

    ✔ 예: 4,000원 태그 → 32분 진행

🚨 자주 발생하는 실수

  • ❌ 다른 기기 투입구에 결제
  • ❌ 고온으로 섬세 의류 건조
  • ❌ 문이 완전히 닫히지 않은 상태에서 시작
  • ✔ 사용 전 한 번 더 확인해주세요.
